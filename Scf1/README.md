- **Journal:** Science
- **Year:** 2023
- **DOI:** [10.1126/science.adf8972](https://doi.org/10.1126/science.adf8972)
- **PMID:** 37769084
- **PMCID:** PMC11235122
- **Original Data:**

## Analysis Overview

This project aims to re-analyze the transcriptomic and genomic data from the Santana et al. 2023 Science paper on the *Candida auris*-specific adhesin Scf1, which governs surface association, colonization, and virulence.

### Key Findings from Original Paper:
- Scf1 is a *C. auris*-specific adhesin
- Governs fungal surface association and biofilm formation
- Critical for colonization and virulence
- Potential therapeutic target

### Re-analysis Goals:
- [ ] Reproduce RNA-seq analysis
- [ ] Validate gene expression findings
- [ ] Explore additional transcriptomic patterns
- [ ] Analyze insertional mutant sequencing data
